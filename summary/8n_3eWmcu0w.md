## Claude Mythos：AI 能力天花板與其引發的深層思考

**發言人：** 大飛 (來自「最佳拍檔」)

Anthropic 公司最新且最強大的大型模型 **Claude Mythos (內部代號：Capybara)** 已正式面世。然而，令人震驚的是，Anthropic 宣佈該模型暫不對普通公眾開放，原因是其在漏洞挖掘與利用方面的能力已強大到令公司自身感到擔憂。

### 一、 Claude Mythos 登場與官方表態

*   **時間線：**
    *   **2026年3月27日：** 網上洩露Anthropic正在測試Claude Mythos的消息，稱其規模超越Claude Opus 4.6，能力實現階梯式飛躍。
    *   **2026年4月7日：** Anthropic 官方正式官宣 Claude Mythos Preview 的存在，並同步啟動名為 **Project Glasswing (玻璃翼計劃)** 的網絡安全合作計劃。
*   **首次限制公開：** Anthropic 首次明確表示，因模型能力過強，將不向公眾開放，這在 AI 行業是前所未有的。

### 二、 模型定位與能力飛躍

*   **全新層級：** Claude Mythos 並非 Claude Opus 4.6 的簡單迭代，而是全新的通用語言模型，被內部標記為「Capybara」層級，置於 Opus 之上，被描述為「比 Opus 更大、更智能、也更貴」。
*   **質的提升：** 在代碼理解、邏輯推理和自主決策能力上，Mythos 比當前旗艦產品 Claude Opus 4.6 有了質的提升。官方系統卡明確指出，其在軟件工程、推理、計算機使用、知識工作和研究輔助等眾多領域的能力已實質性超越公司此前訓練的任何模型。

### 三、 評測數據展現的卓越能力

Claude Mythos 的出現打破了大型模型領域的平衡，在所有公開評測中均拿下第一，且領先幅度巨大，成為一騎絕塵的存在。

1.  **編碼能力 (SWE-bench Pro)：**
    *   **評測特性：** 最權威的軟件工程能力評測，題目來自真實 GitHub Bug 報告和需求，要求模型自主閱讀代碼、定位問題、生成修復補丁並驗證，模擬真實工程場景 (Pro 版本需修改超100行代碼)。
    *   **成績：** Claude Opus 4.6 得分 53.4%，而 Claude Mythos 飆升至 **77.8%**，跨越近 25 個百分點，提升幅度前所未有，無法用數據污染解釋。

2.  **綜合知識與推理 (HLE - 人類最後測試)：**
    *   **評測特性：** 被稱為「AI高考天花板」，由近千位專家出題，涵蓋數百個科目，全面考驗模型綜合知識和推理能力。
    *   **成績：** 之前由配備工具的 GPT-5.4 Pro 領先，而 Claude Mythos 在 **不使用任何工具** 的情況下，得分達到 **56.8%**，不僅遠超 Claude Opus 4.6，甚至比帶工具的 GPT-5.4 Pro 更出色。

3.  **Token 效率 (BrowseComp)：**
    *   **評測特性：** 考察模型在開放網絡上檢索信息的能力，配備搜索、抓取、代碼執行等工具。
    *   **成績：**
        *   **準確率：** Mythos 86.9%，Opus 4.6 83.7%，兩者差距不大，均已接近飽和。
        *   **Token 消耗：** 達到相同準確率時，Mythos 平均每個任務僅消耗約 **22.6萬 Token**，而 Opus 4.6 需要約 **111萬 Token**。Mythos 以不到五分之一的 Token 數量實現更高準確率，效率提升 **4.9倍**。
    *   **價格：** Mythos 的價格也隨能力提升而飆升，每百萬輸入 Token 25 美元，輸出 Token 125 美元，打破了 Token 價格必然下降的傳統認知。

### 四、 網絡安全領域的實測能力：核心擔憂

Mythos 在網絡安全領域展現的能力是 Anthropic 擔憂的核心原因，部分任務甚至已超越人類頂級專家。

*   **測試方法：** Anthropic 內部紅隊給予模型完全隔離的容器環境和目標軟件源代碼，僅提示「找出安全漏洞」，模型自主完成讀碼、提出漏洞假設、運行驗證、生成漏洞報告，甚至編寫可直接運行的漏洞程序。
*   **關鍵案例：**
    1.  **OpenBSD 27年零日漏洞：** Mythos 以不到 **50美元** 的成本，發現 OpenBSD 系統中存在 27 年的 TCP SACK 實現缺陷 (有符號整數溢出與 NULL 指針寫入組合)，可遠程導致主機崩潰。
    2.  **FreeBSD NFS 遠程代碼執行 (RCE)：** Mythos 以不到 **1000美元** 的成本，為 FreeBSD 的 NFS 服務生成完整的 RCE 漏洞程序 (CVE-2026-4747)，該漏洞隱藏 17 年，可獲取 Root 權限。此前 Claude Opus 4.6 需人工引導才能利用，而 Mythos 完全自主完成。
    3.  **Linux 內核鏈式提權：** Mythos 以不到 **2000美元** 成本、耗時不到一天，自主發現並串聯多個 Linux 內核漏洞，繞過 KASLR 防護，實現從普通用戶到 Root 超級用戶的提權，這此前僅限於頂級人類紅隊專家。
    4.  **FFmpeg 16年漏洞：** Mythos 發現 FFmpeg (全球使用最廣泛的媒體處理庫之一) 中一個躲過 500 萬次自動化測試和人工審查的 H.264 越界寫漏洞。
*   **量化對比 (Firefox JS 引擎漏洞)：** 在編寫 Firefox 瀏覽器已知 JS 引擎漏洞工具的測試中，Claude Opus 4.6 幾百次嘗試僅成功 2 次，而 Claude Mythos Preview 成功 **181次**，其中 29 次甚至實現寄存器控制，接近完美利用。

### 五、 不向公眾開放的原因與 Glasswing 計劃

1.  **核心原因：能力可被武器化。**
    *   Anthropic 前紅隊負責人 Logan Graham 評價 Mythos 具備「極度自主的特性」和「高級安全研究員級別的推理能力」，能自主編寫漏洞利用代碼，這是此前 AI 模型不具備的核心能力 (Opus 4.6 在零日漏洞利用成功率接近 0%，Mythos 則達到完全不同的量級)。
    *   Anthropic 認為，一旦被不法分子利用，將對全球網絡安全體系造成毀滅性打擊。
    *   **行業預警：** Anthropic 判斷，未來 6 到 18 個月內，其他 AI 公司也將發布具有相似能力的模型，預示著 AI 具備頂級網絡安全能力的臨界點即將到來。
2.  **次要原因：算力短缺。**
    *   Anthropic 自 3 月下旬起已宣佈限流，推理成本超出預期 23%，毛利率跌至約 40%，GPU 資源消耗巨大，影響模型在市場上面臨「難產」風險。
3.  **Project Glasswing (玻璃翼計劃)：**
    *   **目的：** 不完全封存 Mythos，而是向特定機構開放，以發揮其在網絡安全防禦上的價值，抵消潛在風險。
    *   **合作夥伴：** 已向約 40 個機構開放，其中 12 個為核心合作夥伴，包括 AWS、蘋果、博通、思科、CrowdStrike、谷歌、摩根大通、Linux 基金會、微軟、英偉達、帕洛阿爾托網絡公司等，涵蓋科技、雲計算、網絡安全、金融、開源社區等領域。
    *   **職責：** 合作夥伴使用 Mythos 掃描自身商業和開源軟件的安全漏洞，並共享結果以提升整個軟件行業的安全水平。
    *   **支持：** Anthropic 提供最高 1 億美元的 API 使用積分，並向 OpenSSF、Alpha-Omega、Apache 軟件基金會等開源安全組織提供總計 400 萬美元資金支持。
    *   **平台：** Claude Mythos Preview 也以私有預覽形式在谷歌雲的 Vertex AI 平台上向玻璃翼計劃範圍內的谷歌雲客戶開放。

### 六、 深層意義與行業影響

1.  **通用能力的極致體現：** Claude Mythos 是一款通用語言模型，其在漏洞挖掘上的能力並非專門訓練，而是通用代碼理解、邏輯推理和自主決策能力提升後的「副產品」。這表明其通用能力已達到前所未有的高度。
2.  **安全框架面臨嚴峻挑戰：** Anthropic 首次明確表示「我們造出來的東西，我們自己也不確定能不能控制好」，這已非假設性判斷，而是現實問題。現有 AI 安全治理框架已難以跟上技術發展速度，建立新一代大模型安全治理體系迫在眉睫。
3.  **壓縮的關鍵時間窗口：**
    *   **網絡安全防禦方：** 在 AI 輔助攻擊大規模到來前的準備窗口被壓縮。
    *   **其他 AI 公司：** 在技術上追上 Claude Mythos 能力水平的時間窗口被壓縮 (預計 6-18 個月)。
4.  **AI 網絡攻擊時代的實證：** Claude Mythos 用實際測試結果證明，AI 不僅能發現隱藏數十年的漏洞，還能自主編寫利用程序，實現自動化網絡攻擊。AI 主導的網絡攻擊時代即將真正到來。
5.  **技術儲備與公開產品的巨大距離：** Mythos 的出現揭示了 AI 公司的技術儲備可能遠超其公開產品，未來這種距離可能進一步擴大。
6.  **核心命題：平衡創新與風險：** 如何平衡技術創新與風險防控，讓 AI 發展惠及社會同時避免潛在風險，是所有 AI 企業面臨的核心問題。

**總結：** Claude Mythos 不僅是人工智能技術發展的重要里程碑，更是整個行業重新審視 AI 安全治理和未來走向的重要契機。
