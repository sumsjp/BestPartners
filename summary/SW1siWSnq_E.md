**獨特的結構與價值**

*   此結構讓非營利體能夠持續增長價值，PBC 則能不斷吸收資本實現商業化。
*   OpenAI 一開始就持有約 1300 億美元的股權，計畫首先將 250 億美元投入健康、AI 安全與社會韌性等方向，使 AGI 造福全人類的目標有實際資金支持。
*   Altman 認為資本主義仍是最有效的價值創造機制，非營利基金補全市場無法高效覆蓋的領域，AI 將加速科學發現、推動科研數據開放。
*   Nadella 視此雙層結構為企業與社會利益之間的橋樑，微軟獲得分紅權與分成收益，更重要的是在核心算力、模型、專利及 API 實現長期綁定。
*   微軟等於擁有前沿模型的免費授權，可在 Github Copilot、Microsoft 365 Copilot 等產品中無縫嵌入 OpenAI 模型，形成持續的價值循環。

**共生關係與雲端轉型**

*   雙方在算力、模型、分成、生態四個層面形成共生關係。微軟提供計算基礎設施，OpenAI 釋放模型價值。
*   此合作模式已成為雲計算 2.0 時代的標準模板。
*   由於獨家託管 OpenAI 的主力模型，微軟吸引大量企業客戶從 AWS 遷移到 Azure。
*   Azure、GitHub、Copilot、ChatGPT 四個產品聯動，構成地球上最大的 AI 產品體系。

**比爾·蓋茲的轉變**

*   最初連比爾·蓋茲也對此合作持有懷疑態度，直到他看到了 GPT-4 的演示之後，態度才徹底轉向。

**算力投資與收入模式**

*   OpenAI 的年營收約為 130 億美元，卻計畫在未來五年投入 1.4 兆美元的算力建設。

    1.  成為全球最重要的 AI 雲服務提供商。
    2.  發展面向消費者的 AI 設備。
    3.  推動 AI 科學家專案，實現自動化科研。
*   Nadella 認同此觀點，並指出 OpenAI 的執行力幾乎沒有失手，這是微軟持續加碼的根本原因。

**算力約束與能源需求**

*   Altman 將算力問題類比為能源經濟學，認為算力的需求不能脫離成本曲線來討論。
*   Nadella 接著指出現在最緊缺的不是晶片，而是電力。AI 面臨的最大問題不是算力不足，而是無法快速獲得足夠的電能。
*   未來兩到三年內幾乎不可能出現算力過剩，真正的競爭將集中在能否最快建好接近電源的資料中心。

**SaaS 架構的重寫**

*   Nadella 指出，過去軟體是資料 + 邏輯 + 界面的三層耦合結構，而現在 AI 不再尊重這種耦合，新的邏輯層被 Agent 所取代。
*   傳統的 SaaS 應用像是一個固定流程的系統，而 Agent 是一種具備回饋迴圈的生產單元。
*   新一代 SaaS 的核心不再是寫死的邏輯，而是實時優化的 Agent。

*   Agent 將會是新的席位（Seat），每一個 Agent 都能代表一個工作的完成。
*   AI 軟體的利潤結構將回到正循環，未來的軟體價值不再取決於訂閱數量，而取決於 Agent 的完成效率。
