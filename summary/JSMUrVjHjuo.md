**Andrej Karpathy 深度訪談：AI革命的“深水區”與我們的未來**

在最新一期播客中，OpenAI前聯合創始人、AI領域頂尖專家Andrej Karpathy（安德烈·卡帕西）坦言，因AI的飛速發展，他陷入了某種“AI精神錯亂”。他毫無保留地分享了對AI編程、大模型發展、開源閉源競爭、以及AI如何重塑教育和就業的核心思考。卡帕西認為，AI革命已從模型研發步入**場景落地的“深水區”**，我們每個人都在被這場變革所改變。

---

### **一、AI編程範式的顛覆性巨變：Agent主導**

1.  **工作流重寫：** Karpathy直言，自2025年12月起，軟件工程的工作流已被Agent徹底改寫。他本人已幾乎不再親自編寫代碼，從過去80%手寫、20%AI輔助，轉變為現在**80%甚至更多由Agent完成**。
2.  **人類角色轉變：** 人類從“寫代碼的人”變成了“用自然語言向Agent表達想法的人”。這已成為整個行業的常態，甚至有團隊工程師完全不手寫代碼，只通過語音指令與Agent交互。
3.  **“AI精神錯亂”的根源：** Agent使個體能實現的事情呈指數級增長。過去創造力受限於打字速度、技術儲備，如今只要能精準表達需求，就能實現超乎想像的功能。這種極速變化讓Karpathy每天都在探索Agent的能力邊界，以及如何多Agent協同工作，稍有跟不上前沿便產生強烈焦慮。
4.  **Agent操作是關鍵：** 很多人用Agent效果不佳，Karpathy認為是人類未掌握好操作技巧（Skill），而非模型本身能力問題。例如，指令不夠精準、未配置記憶工具、或不懂如何讓Agent并行工作。
5.  **宏觀操作思維：** 以OpenClaw作者彼得·斯坦伯格為例，他讓多個Codex Agent并行處理10個代碼庫的檢查工作，人類僅需審查產出。這種“宏觀操作”思路讓人類跳出細節，轉而負責分配任務、把控方向，這也是當前AI從業者的核心必修課。

### **二、行業焦慮點轉移：從GPU算力到Token吞吐量**

1.  **新焦慮：Token未用完！** Karpathy分享的第二個核心觀點是，AI行業的焦慮點已從“能不能跑滿GPU算力”轉向**“能不能用完Token”**。他甚至直言：“訂閱沒用完我就焦慮，說明我的Token吞吐量沒拉滿。”
2.  **指標變化原因：** AI能力實現躍升，算力不再是核心瓶頸，人類的操作能力才是。你的Token吞吐量是多少？你能指揮多少Token同時運行？這成為衡量AI從業者能力關鍵。
3.  **正向循環：** 這種焦慮是好事，它會推動人類不斷提升操作技能（Skill），進而解鎖Agent更多能力，形成“Skill提升→能力解鎖→效率提升”的正向循環。
4.  **未來方向：** 必然是從單次與Agent聊天，走向多Agent協作、團隊配合，甚至實現Agent的自主循環運行。

### **三、Agent作為“數字世界的粘合劑”：家庭自動化案例**

1.  **“多比”家庭Agent：** Karpathy用OpenClaw打造了名為“多比”的家庭Agent，它徹底接管了他的家庭（Sonos音箱、燈光、空調、窗簾、安保系統等）。
2.  **核心結論：App應是Agent可調用的API！** 多比通過IP掃描、逆向工程、網絡搜索自動發現設備API並進行控制，實現了全屋智能聯動。卡帕西深刻感受到，應用商店裡那些配套智能家居的定製App其實沒有必要，設備只需開放API讓Agent直接調用。
3.  **“Agent化的網絡”：** 未來的客戶不再只是人類，還有代表人類行事的Agent。這意味著整個行業正走向徹底的重構。
4.  **安全與隱私：** 雖然多比能力強大，但Karpathy目前並未開放其郵件、日曆等全部訪問權限，因早期Agent技術尚粗糙，存在風險。
5.  **未來願景：** 他堅信，一兩年或三年後，這種家庭自動化能力會成為**免費的基礎服務**，無需人類專業操作。
6.  **Agent的魅力：** OpenClaw打動用戶的核心不在於功能最強，而是它更接近人類心目中的AI形態——有人格、有記憶，能通過單一入口實現所有功能，而非冰冷的Token生成器。

### **四、自動研究（AutoResearch）的崛起：將人類從研究瓶頸中移除**

1.  **目標：Agent自主運行研究：** Karpathy當前最關注的方向是AutoResearch。他希望將人類從研究瓶頸中徹底移除，人類只需安排好任務、設定目標、指標和行為邊界，讓Agent完全自主運行。
2.  **專家經驗被超越：** Karpathy以其經典的nanoGPT項目為例。作為擁有20年研究經驗的專家，他已對該項目進行了極致手工調優，自認達到性能極限。然而，自動研究系統運行一晚後，竟然發現了多個他忽略的調優空間（如值嵌入的權重衰減、Adam beta參數），證明AI在處理有客觀指標、可量化的研究任務時，能力已超越人類。
3.  **未來研究流程：** 將徹底改變。點子隊列將自動接收來自arXiv論文、GitHub倉庫的靈感，或人類研究員的貢獻，由自動化Agent抓取任務並嘗試。可行代碼進入功能分支，人類只需偶爾監控並合併。核心是**將人從研究的所有流程中撤走，追求極高的Token吞吐量**。

### **五、AI能力分布的“鋸齒狀”特性與模型分化**

1.  **“天才與10歲小孩”：** Karpathy形象地比喻與AI對話，感覺像是同時與一位天才程序員和一個10歲小孩對話——在某些領域（如CUDA算子、超參數優化）能力極強，但在另一些領域（如理解人類細微意圖、講笑話）卻表現幼稚。
2.  **原因：強化學習訓練邏輯：** 大模型主要通過強化學習訓練，實驗室只能在可驗證、有獎勵反饋的領域提升模型能力。沒有明確標準的領域，能力就會明顯落後。
3.  **模型“物種分化”：** Karpathy認為，未來大模型會出現更多的“物種分化”，而不是追求一個在所有領域都聰明的單一大模型。會出現更多專用的小模型，在特定領域被極致優化。
4.  **當前瓶頸：** 目前物種分化尚未主流，原因包括：前沿實驗室仍在追求包羅萬象的通用大模型、核心技術（如不損通用能力的微調、持續學習、修改模型權重）不成熟，以及算力成本考量。

### **六、AI研究的并行化與“Auto Research @ Home”**

1.  **從單線程到并行化：** AI研究正在從過去單個研究者不斷嘗試的“單線程”模式，走向多個自動研究員協同、甚至引入互聯網上大量不被信任的閒散算力參與的“并行化”模式。
2.  **“生成極貴，驗證極便宜”：** 利用“生成候選代碼耗費算力，驗證代碼有效性成本低”的特性，可引入不被信任的閒散算力生成候選代碼，再由可信任節點驗證。
3.  **“Auto Research @ home”：** 這種模式與SETI@home等項目高度相似，讓互聯網閒散算力參與科學研究。Karpathy認為其潛力巨大，因為地球上不被信任的閒散算力總量遠超前沿實驗室。
4.  **研究普惠化：** 該模式能讓AI研究變得更加普惠，算力成為一種可以貢獻的資源。未來人們可能會像在乎美元一樣在乎FLOPS，算力將成為真正稀缺和主導的資源。

### **七、AI對就業與產業發展的思考**

1.  **AI是“數字世界的幽靈”：** Karpathy分析美國勞工統計局數據後發現，當前主流AI更多是數字世界的幽靈，高效操縱數字信息，但還未真正物理具身。物理世界的操作遠比數字世界慢。
2.  **數字世界先爆發：** AI會先引發數字世界的爆炸式變化，重構大量數字信息處理工作。物理世界會相對滯後，需要巨量資本、時間和信念。
3.  **就業建議：** 面臨就業挑戰者，應盡量跟上AI工具發展，保持好奇，學會與AI協作，將AI視為強大的生產力工具。
4.  **軟件工程前景：** Karpathy態度謹慎樂觀。長期看，前沿AI實驗室的研究員正在“光榮地把自己自動化掉”——他們的核心工作就是打造能取代自己的AI系統。他觀察到許多研究員已出現“AI精神錯亂”，因為他們親手打造著自己的替代品。

### **八、開源與閉源的競爭格局：權力平衡的關鍵**

1.  **差距縮小：** 閉源模型目前仍領先，但與開源模型的差距已縮小到6-8個月左右。
2.  **開源的必要性：** Karpathy作為長期支持開源的研究者，認為行業需要一個開源、安全可信的共同工作空間（如操作系統領域的Linux）。開源模型的存在能有效避免完全封閉智能帶來的系統性風險，實現行業權力平衡。
3.  **未來格局：** 閉源模型服務於對最前沿智能有需求的高難度任務（如諾貝爾獎級科學研究），而開源模型則會覆蓋絕大多數消費級場景和基礎用例，甚至能在本地運行，成為行業基礎。
4.  **擔憂：** Karpathy擔憂閉源模型頭部越來越集中的趨勢，希望有更多前沿AI實驗室和團隊參與大模型研發，因為多組人共同思考難題總比少數人關起門來決定要好。

### **九、AI在物理世界：數物接口是關鍵**

1.  **潛力更大但滯後：** AI在物理世界的規模化應用會明顯滯後於數字世界，但其總潛在市場可能比純數字世界更大。
2.  **發展時間線：** 首先是數字世界大爆發，重構所有數字信息處理工作；然後延伸到**數字與物理的接口部分**（傳感器和執行器），因為純數字任務終會做完，AI需從物理世界獲取數據、做實驗來學習；最後才是物理世界的規模化應用。
3.  **“數物接口”公司：** 目前已有公司在探索，如Periodic公司用AI做材料科學自動研究（昂貴實驗室設備作為傳感器）；也有公司將人類作為傳感器，付費采集物理世界數據。
4.  **Agent化Web：** 未來會出現發達的信息市場，人們可以付費讓Agent在特定地點獲取數據並進行分析決策。Agent化的網絡目前處於早期，但必然是未來方向。

### **十、AI重構教育模式：人類互相教授知識的時代將結束**

1.  **MicroGPT的啟示：** Karpathy的個人項目microGPT，將大語言模型簡化到200行Python代碼，揭示了教育的未來。
2.  **Agent教授知識：** 他認為，以前他會想錄視頻講解代碼，但現在Agent可以從各種角度解釋這些簡單代碼，並能根據用戶語言水平無限耐心地反復講解。人類反而能從Agent那裡學得更好。
3.  **人類角色轉變：** Karpathy可以設計課程骨架，讓Agent負責執行知識傳遞。
4.  **顛覆性結論：** **“人類互相教授知識的時代，快要結束了。”** 未來的教育模式將是：先讓Agent搞懂知識，然後讓Agent來教人。這是一場人類必輸的戰鬥。
5.  **人類的核心價值：** 未來人類需要戰略性地把時間花在Agent做不到的事情上，例如創造像microGPT這樣的核心成果，這才是人類的核心價值所在。

**總結：**

Karpathy的核心觀點可以概括為一句話：**AI的革命，已經從模型研發進入了場景落地和生態重構的階段，Agent成為了核心驅動力，正以我們無法想像的速度改變世界。** 我們每個人都是這場變革的參與者和見證者，面對AI的“精神錯亂”將可能成為常態，我們需要主動適應和參與這場重塑。
