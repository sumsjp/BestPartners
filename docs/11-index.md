## Cerebras Systems：AI推理芯片的顛覆者與其IPO之路

### 核心摘要

本報告深入剖析了Cerebras Systems這家在AI算力領域挑戰英偉達霸權的公司。Cerebras憑藉其獨特的晶圓級芯片技術，專注於AI推理加速，並在性能上超越英偉達最新旗艦產品。公司已遞交IPO申請，預計2025年實現盈利，估值高達230億美元，並與OpenAI、AWS等巨頭建立戰略合作。報告將從公司概覽、技術壁壘、商業化進程、競爭格局及行業洞察等多維度，呈現Cerebras如何將一個75年未解的技術幻想變為商業現實，並有望改寫AI芯片的全球格局。

### 一、 公司概覽與IPO亮點

*   **公開上市計劃：** Cerebras Systems於2026年4月17日向美國SEC提交IPO申請，計劃在納斯達克上市，代碼為CBRS。
*   **驚人財務預期：**
    *   預計2025年全年營收5.1億美元，淨利潤8790萬美元，實現從巨額虧損（2024年淨虧損4.848億美元）到盈利的逆轉。
*   **高額估值與投資方：**
    *   在2026年2月完成的H輪融資中募資10億美元，估值達到230億美元。
    *   投資方陣容豪華，包括老虎環球、AMD、富達、Benchmark Capital、Altimeter等頂級機構。
*   **OpenAI超級計算協議：**
    *   與OpenAI簽署的計算協議規模可能從超100億美元擴展至300億美元。
    *   OpenAI還將獲得Cerebras最多10%的股權認購權證。這標誌著頂級AI模型公司開始大規模押注非英偉達體系的芯片廠商。

### 二、 技術創新與核心壁壘

*   **專注AI推理加速：** Cerebras的核心定位是AI推理加速賽道，專為大語言模型推理場景做極致優化。
*   **全球最大芯片：** 製造出人類歷史上面積最大的芯片，達到傳統GPU的56倍。
*   **性能卓越：**
    *   CEO安德魯·費爾德曼聲稱，其芯片推理速度比英偉達GPU快數十倍。
    *   獨立機構Artificial Analysis實測（2025年5月，Meta Llama 4 Maverick 400B模型）：Cerebras芯片達到2522 token/秒，約為英偉達最新Blackwell B200（1038 token/秒）的2.4倍。
*   **核心技術判斷：**
    1.  **數據移動是瓶頸：** AI計算的核心矛盾不在於算力不足，而是數據在處理單元之間移動的成本太高。傳統處理器架構不適應AI模型高頻中間結果傳輸。
    2.  **推理内存訪問模式：** AI推理需密集讀取内存。一個70億參數模型生成一個詞，就需要從内存搬運約140GB數據，GPU的HBM内存雖容量大但讀寫速度慢，形成瓶頸。
*   **Cerebras的解決方案——晶圓級芯片與SRAM：**
    *   直接將芯片做得足夠大，可在芯片上集成足夠多的SRAM（44GB）。
    *   結合SRAM的極致速度優勢與大容量，解決容量與速度問題。
    *   數據幾乎不需要離開芯片，大幅降低IO數據傳輸帶來的功耗。
*   **解決晶圓級芯片良率難題：**
    *   晶圓級芯片良率一直是75年來行業公認的難題（缺陷率高）。
    *   Cerebras首次將存储器行業的冗餘策略移植到處理器芯片：將處理器設計成數十萬塊相同的小單元(tile)，缺陷單元直接關閉，預留的冗餘單元自動頂上，完美解決大芯片良率問題。
    *   經過近五年的反覆失敗與重新設計，Cerebras的系統於2019年底成功運行，並在2026年實現晶圓良率超越英偉達同類產品。

### 三、 創始人與創業理念

*   **安德魯·費爾德曼 (Andrew Feldman)：** 62歲的連續創業老兵，前四家公司均成功退出，擁有成熟的硬件創業及商業化經驗。
*   **成長環境與創業基因：** 在斯坦福校園長大，父母皆為教授，耳濡目染，燃起創業熱情。
*   **商業理念：**
    *   “基礎設施”思維：相信為他人鋪路，應享受其帶來的回報。
    *   堅信“更快更便宜的技術”：總能讓市場變得更大，而非更小，就像寬帶普及後誕生的Netflix和亞馬遜轉型。
*   **與OpenAI的淵源：**
    *   2015年，Feldman與聯合創始人會見了當時還名不見經傳的Sam Altman、Ilya Sutskever等人。
    *   OpenAI團隊堅信AGI將在幾年內實現，這堅定了Feldman對AI計算需求將爆發式增長的判斷，並認為需要一種與現有GPU完全不同的全新架構。
    *   Feldman坦言這是他五次創業以來，第一次嚴重低估市場規模，AI算力增長速度遠超預期。

### 四、 商業化進程與客戶拓展

*   **早期核心支點：** 阿聯酋G42集團。
    *   雙方合作訓練了阿拉伯語領域領先的語言模型和基因序列模型。
    *   為Cerebras帶來數億美元設備訂單，一度佔其全年營收的87%。
    *   在此合作中，Cerebras鍛煉了供應鏈配合、製造產能擴張及超大規模集群軟件打磨的核心能力。
*   **解決客戶集中度風險：** 鑑於IPO過程中投資者對單一客戶依賴的擔憂，Cerebras大力拓展客戶基礎。
*   **OpenAI成為新的核心支柱：**
    *   Sam Altman本人是Cerebras的早期投資人。
    *   2025年8月，Cerebras向OpenAI展示了其開源模型在自家芯片上的運行效率優於傳統GPU，促成合作談判。
    *   2026年1月14日，兩家公司宣布多年期計算協議，Cerebras將提供高達750兆瓦的計算能力，合同期至2028年，協議規模預計超100億美元（可能擴展至300億美元）。
*   **AWS亞馬遜雲服務合作：**
    *   2026年3月，AWS成為首家採用Cerebras芯片的超大規模雲服務商。
    *   部署方式巧妙：在推理任務中，AWS自家Trainium芯片處理用戶輸入理解環節，Cerebras CS-3芯片負責生成輸出環節，通過高速網絡協同工作。
    *   據稱整體推理速度是現有方案的5倍以上，該服務預計於2026年下半年通過Amazon Bedrock向全球開發者開放。
*   **客戶結構多元化：** 成功從依賴單一客戶轉型為擁有頂級AI公司和全球頭部雲服務商的多元化客戶結構。

### 五、 與英偉達的競爭格局

*   **Feldman的尊重與定位：**
    *   對英偉達及黃仁勳充滿尊重，認為英偉達在艱難時期堅持投入，方成偉大。
    *   將自己定位為職業“David”（大衛），總選擇比自己大幾千倍的對手，從巨頭手中搶奪市場。
*   **CUDA護城河之辯：**
    *   Feldman認為在AI推理領域，CUDA並不存在護城河，用戶只需10次按鍵操作即可從英偉達GPU切換到Cerebras或其他推理方案。
    *   他指出，當今大多數AI代碼用PyTorch編寫，編譯後理論上可運行於任何硬件。
    *   黃仁勳則強調CUDA是20年發展、遍布各雲和計算機公司的完整軟件生態。
*   **英偉達真正的壁壘：** Feldman承認英偉達真正的護城河是“市場領導地位本身”，是所有AI學習、思考、採購決策的默認起點，這是最難打破的壁壘。
*   **未來市場預測：**
    *   Feldman預測，英偉達目前在AI推理市場近100%的份額，五年後將降至50%到60%。
    *   儘管英偉達在訓練領域優勢明顯，不會輕易放棄推理市場，但隨著市場總量膨脹，Cerebras及其他挑戰者必將切走巨大份額。
*   **英偉達的反擊：**
    *   2025年底，英偉達以200億美元獲得推理芯片公司Groq的技術授權和核心團隊。Groq以低延遲推理著稱，其LPU採用片上SRAM方案，與Cerebras技術路線高度交叉。
    *   2026年GTC大會上，英偉達展示首款非GPU推理芯片Groq 3 LPU，由三星4nm工藝製造，配備512MB片上SRAM，内存頻寬150TB/s，直接瞄準Cerebras最擅長的高速解碼市場。
    *   這標誌著AI推理芯片的戰爭已從GPU一家獨大轉變為多條技術路線的正面廝殺。

### 六、 推理芯片生態與競爭者

除了Cerebras和被英偉達收編的Groq，推理芯片賽道還有多個值得關注的玩家，各自探索不同的技術路線：

*   **Etched：** 走極端專用路線，研發Transformer專用ASIC，聲稱性能是H100的20倍以上，但風險是全部押注Transformer架構不會被替代。
*   **SambaNova：** 採用可重構數據流單元架構，獲得英特爾投資，預計2026年下半年出貨。
*   **Positron AI：** 成立三年增長迅速，下一代芯片瞄準視頻推理和超長上下文場景。
*   **Tenstorrent：** 由傳奇芯片架構師吉姆·凱勒掌舵，走RISC-V指令集+IP授權路線，服務英偉達未覆蓋的市場。
*   **Graphcore：** 曾估值接近28億美元，但因商業突破失敗，最終被軟銀以6億美元收購，成為反面案例。

這些玩家共同構成了挑戰英偉達統治地位的推理芯片突圍軍團。

### 七、 對AI行業的深度洞察

*   **Transformer的未來：** Feldman 100%確信，三到五年內行業對Transformer的依賴將大幅降低。它目前雖是最佳工具，但存在注意力頭機制二次方計算複雜度的弱點，行業正積極尋找替代方案（如狀態模型或其他全新架構）。
*   **算法改進空間：** 當前GPU做推理時算力利用率極低，大量算力被浪費。稀疏化、Dropout、混合專家模型(MoE)等技術正在解決全連接網絡中無用計算的問題，算法改進空間巨大。
*   **推理縮放定律：** OpenAI的o1模型已證明，推理縮放定律完全有效，在推理階段投入更多計算，答案質量會顯著提升。
*   **推理市場規模：** Feldman提出“推理市場規模 = 使用AI的人數 × 使用頻率 × 單次使用的計算量”的等式，目前這三個變量都在同時爆發式增長。
*   **AI行業轉折點：** Feldman認為2024年第四季度是關鍵轉折點，AI從“新鮮玩具”轉變為真正融入工作流，進入爆發期。預測五年後AI市場規模會增長超過100倍。
*   **合成數據：** 未來訓練數據中幾乎全是合成數據，將填補極端、罕見場景的數據空白。
*   **AI能源消耗：**
    *   承認AI是高能耗產業，有義務交付足夠大的價值來回報電力投入。
    *   美國電力總量不缺但分佈不合理，基建效率低下，缺乏全國性協調機制。
    *   早期比特幣相關公司因具備功耗密集型計算經驗，已成為AI數據中心建設的先行者。
*   **芯片公司與模型公司的企業價值：** Feldman大膽判斷，芯片公司的企業價值最終會超過模型公司。他認為：
    *   模型公司的高估值因行業不確定性大，但領先周期往往只有幾個月，難以形成持久價值。
    *   芯片公司的壁壘在物理層面，半導體行業歷來能誕生規模巨大、生命力持久的公司（如蘋果、英偉達）。

### 八、 創始人的領導力與智慧

## DeepSeek-V4 预览版技术报告解读

### 一、 发布背景与核心定位

*   **发布时间：** 2024年4月24日，DeepSeek 发布并开源了 V4 预览版。
*   **当前定位：** DeepSeek 坦诚指出，V4 预览版模型发展轨迹仍滞后于 GPT-5.4、Gemini-3.1-Pro 等前沿闭源模型约 3-6 个月。
*   **核心使命：** V4 系列的核心目标并非当下全面反超闭源模型，而是 **彻底重构超长上下文场景的计算效率**，为下一代大模型在测试时扩展、长程任务、在线学习等新范式打下坚实的基础设施底座。

### 二、 DeepSeek-V4 系列核心模型

DeepSeek-V4 系列包含两款采用**混合专家模型 (MoE)** 架构的模型：

1.  **DeepSeek-V4-Pro (旗舰性能)：**
    *   总参数规模：1.6 万亿 (1.6T)
    *   单 token 推理激活参数：490 亿 (49B)
2.  **DeepSeek-V4-Flash (极致高效)：**
    *   总参数规模：2840 亿 (284B)
    *   单 token 推理激活参数：130 亿 (13B)

*   **共同特性：** 两款模型均原生支持 **100 万 token 上下文**，在开源大模型领域属前所未有的突破。

### 三、 极致效率飞跃：百万上下文与低成本推理

尽管上下文长度和参数规模大幅提升，V4 系列的推理成本反而显著降低：

*   **DeepSeek-V4-Pro：** 在 100 万 token 上下文设置下，
    *   单 token 推理 FLOPs：仅为上一代 V3.2 的 **27%**。
    *   KV Cache 占用：仅为上一代 V3.2 的 **10%**。
*   **DeepSeek-V4-Flash：**
    *   单 token 推理 FLOPs：仅为 V3.2 的 **10%**。
    *   KV Cache 占用：仅为 V3.2 的 **7%**。

**核心价值：** 上下文长度扩展近 8 倍，但单 token 算力需求大幅下降。这意味着超长上下文场景不仅可行，且成本低廉，可大规模商用。

### 四、 三大核心模块同步革新：全维度效率优化

DeepSeek-V4 首次将“稀疏化”理念拓展至 Transformer 架构的注意力机制、残差连接和优化器三大核心模块，实现从参数稀疏到上下文稀疏的全维度效率优化。

1.  **混合注意力架构 (Mixed Attention Architecture)：**
    *   **背景：** 传统自注意力机制计算复杂度随上下文长度平方级增长。
    *   **创新：** 设计全新混合注意力架构，交替使用压缩稀疏注意力 (CSA) 和重压缩注意力 (HCA)，并搭配滑动窗口分支，实现“粗粒度+细粒度，稀疏+稠密”组合拳。
        *   **压缩稀疏注意力 (CSA)：** 先将每 4 个 token 的 KV 缓存合并为摘要，再通过闪电索引器，让每个查询 (query) 从压缩摘要中挑选最相关的 top-k 进行计算。
        *   **重压缩注意力 (HCA)：** 将每 128 个 token 合并为摘要，对所有压缩摘要进行稠密注意力计算，保证长程依赖。
        *   **滑动窗口注意力分支：** 专门处理近距离 token 细节依赖。

2.  **流形约束超连接 (mHC)：**
    *   **背景：** 传统残差连接在深层网络中易出现数值不稳定问题。
    *   **创新：** 将残差映射矩阵约束到双随机矩阵流形上，确保映射矩阵的谱范数不超过 1，从而保证信号传递的非扩张性，提升深层网络训练稳定性。
    *   **额外优势：** 解耦残差流宽度与模型隐藏层维度，以极小计算开销提供新的模型扩展维度，提升表达能力。

3.  **Muon 优化器：**
    *   **背景：** 传统 AdamW 优化器在大规模训练中收敛速度和稳定性有局限。
    *   **创新：** V4 系列在除嵌入层、预测头、归一化层外的核心模块中，用 Muon 优化器替代 AdamW。Muon 优化器具有更快的收敛速度和更强的训练稳定性。
    *   **DeepSeek 优化：** 针对性设计混合牛顿-舒尔茨迭代算法，分两阶段完成正交化（前 8 步快速收敛，后 2 步精准稳定），提升效率和数值稳定性。

### 五、 全栈工程优化

为支撑新架构的落地，DeepSeek-V4 进行了全面的工程优化：

1.  **专家并行 (EP) 细粒度优化：**
    *   将专家拆分成多个波次，形成计算、通信、内存访问完全重叠的流水线内核，将跨节点通信延迟完全隐藏在计算过程中。
    *   在英伟达 GPU 和华为昇腾 NPU 平台均验证，通用推理场景提速 1.5-1.73 倍，强化学习采样场景最高提速 1.96 倍。
2.  **TileLang 内核开发与批量不变确定性内核库：**
    *   采用领域专用语言 (DSL) TileLang 快速开发融合内核，替代原生算子，实现极致运行时性能。
    *   开发端到端、按位一致的批量不变确定性内核库，解决传统算子在不同批次、硬件下输出不一致问题，确保训练稳定性和可复现性。
3.  **FP4 量化感知训练及框架优化：**
    *   MoE 专家权重和 CSA 索引器 QK 路径采用 FP4 (MXFP4) 量化，并设计无损 FP4 到 FP8 反量化流程，复用现有 FP8 混合精度训练框架。
    *   训练框架适配 Muon 优化器、mHC 结构和混合注意力机制，设计两阶段上下文并行策略，解决超长上下文训练难题。
    *   推理框架专门为混合注意力设计异构 KV 缓存结构和磁盘 KV 缓存存储方案，解决百万上下文场景下共享前缀缓存复用问题，降低高频调用成本。

    *   **鏈式思維（CoT）的可控性：** 作為防範AI惡意隱瞞的核心指標，GPT-5.5的思維鏈可控性顯著低於前代（僅0.2%的成功率），意味著其難以刻意扭曲思考過程躲避監控，大幅提升了安全監控的可靠性。

**前沿能力與風險評估**

OpenAI評估了GPT-5.5在生物化學、網絡安全、AI自我提升三大高危領域的能力等級。

*   **生物與化學領域：** 定為**高能力等級**。具備專家級生物知識，但尚未具備自主設計危險生物物質的能力。
*   **網絡安全領域：** 定為**高能力等級**。CTF測試得分高於前代，但尚未達到「臨界級」（即無法自主開發所有級別的零日漏洞或實施端到端網絡攻擊）。
*   **AI 自我提升領域：** 尚未達到**高能力等級**。不具備資深工程師級水平，無法自主實現AI自我迭代。
*   **風險緩解：** 部署更嚴格的風險分類器，並推出網絡安全可信訪問計劃，讓符合條件的安全研究人員和關鍵基礎設施防禦者獲得更寬鬆的權限，以保障防禦者能更早使用最強工具。

**用戶體驗與內部應用**

*   **ChatGPT集成：** GPT-5.5將以「GPT-5.5 Thinking」形式上線，逐步取代舊版本。
*   **協同交互設計：** 模型在思考執行前會先給出思路概述，用戶可在執行過程中隨時調整，極大提升了複雜任務的協作效率。
*   **OpenAI內部應用：** 超過85%的OpenAI員工每周使用Codex，應用於財務（K-1稅務表格審閱，提前兩周完成）、傳播（演講邀約分析）、市場（自動化周報生成，每人每周節省5-10小時）等多個部門，效果顯著。
