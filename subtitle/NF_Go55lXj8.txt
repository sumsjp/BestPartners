接下来我们再来详细介绍一下Claude 3.7 Sonnet
Anthropic 表示
这次开发 Claude 3.7 Sonnet 的理念与市面上其他的推理模型都不一样
正如人类会使用单个大脑进行快速反应和深度思考一样
Anthropic 认为推理应该体现前沿模型的综合能力
而不再是多个完全独立的模型
这种统一的方法将为用户创造更为无缝的体验
按照这个理念
Claude 3.7 Sonnet 形成了很多独有的优势
首先
Claude 3.7 Sonnet 既是一个普通的大语言模型
又是推理模型
你可以选择什么时候希望模型正常回答
什么时候希望它在回答之前思考更长时间
在标准模式下
Claude 3.7 Sonnet 是前代 Claude 3.5 Sonnet 的升级版
在扩展思维模式下
它会在回答之前进行自我反思
从而提高在数学、物理、指令遵循、编码和许多其他任务上的表现
Anthropic 还发现，在两种模式下
模型提示词的工作方式是类似的
其次
当通过 API 使用 Claude 3.7 Sonnet 的时候
用户还可以控制思考预算
你可以告诉 Claude 思考不要超过多少个 token
如果没有指定
那么输出会限制为 128K 个 token
这样方便用户在速度、成本和答案质量之间进行权衡
第三，在开发自家的推理模型时
Anthropic 对数学和计算机科学竞赛问题方面的优化较少
而是将重点转向了更能反映企业实际使用模型方式的现实任务
